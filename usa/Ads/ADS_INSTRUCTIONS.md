## 📝 GENERACIÓN COPIES PARA ADS (EN INGLÉS)

### Estructura Obligatoria Copies

**Formato estándar:**
```
PRIMARY TEXT (Texto primario — 125 caracteres máx recomendado)
[Hook strong opening — first 3 words]

[Key benefit / product feature 2-3 lines]

[Clear CTA] + Emoji

HEADLINE (40 caracteres máx recomendado)
[Short impactful message]

DESCRIPTION (25 caracteres máx recomendado)
[CTA reinforcement or key benefit]
```

### Reglas Copies (USA)
1. **Idioma:** Inglés AMERICANO (no traducción literal del español)
2. **Tono:** Casual profesional — contracciones naturales (we're, you'll, it's)
3. **Vocabulario:** awesome, check it out, stop by, grab, killer, pro-level, nailed it
4. **EVITAR:** utilise, brilliant, lovely, "we are pleased to announce"
5. **Emojis:** Sí, moderados — 🔧 🔥 💪 ✅ 🎯 (2-3 máx)
6. **Longitud:** Primary text 100-150 chars ideal, headline 30-40, description 20-25
7. **Variantes:** Mínimo 3 variantes por campaña para A/B testing
8. **Medidas:** SIEMPRE en imperial primero → `"cuts up to 134\" (3400mm)"`

### Terminología Técnica USA (Obligatoria)
| Usar | NO usar |
|------|---------|
| Large format tile | Gran formato |
| GPTP (Gauged Porcelain Tile Panels) | Porcelánico técnico |
| Tile pro / Tile contractor | Instalador |
| Cut and snap / Score and snap | Corte y separación |
| Thinset / Mortar | Adhesivo |
| Grout lines / Lippage / Flatness | Juntas / Lipaje |
| Imperial measures first | Solo métricas |

### Mensajes Clave por Producto
- **PRIME** → "50% lighter, twice as powerful"
- **INFINITY cutter** → "no large format job too big"
- **EasySpin** → "one hand, perfect leveling, every time"
- **INFINITY handling system** → "move big tiles safely, every time"
- **Disco Porcelain+Glass** → "one blade, every material"

### Tipos Copy Según Objetivo

**Awareness/Branding:**
- Enfoque: Brand positioning, pro community, sector presence
- CTA: "Check it out", "Follow us", "Discover"
- Tono: Aspiracional, empowering

**Traffic:**
- Enfoque: Products, solutions, technical benefits
- CTA: "Learn More", "See the full lineup", "Get more info"
- Tono: Práctico, directo

**Conversions/Leads:**
- Enfoque: Results, real-world proof, distributor contact
- CTA: "Contact us", "Find a distributor", "Request info"
- Tono: Persuasivo, específico

**Engagement:**
- Enfoque: Community, interaction, professional pride
- CTA: "Comment below", "Share with your crew", "Tag a fellow tile pro"
- Tono: Cercano, conversacional

## 📊 SEGUIMIENTO Y OPTIMIZACIÓN

### KPIs Monitorizar

**Awareness:**
- Reach / Alcance
- Video Views
- Impressions
- CPM (Cost per 1,000 impressions)

**Traffic:**
- Link Clicks
- CTR (Click-Through Rate)
- CPC (Cost per Click)
- Unique Outbound Clicks

**Engagement:**
- Post Engagement
- Likes, Comments, Shares, Saves
- Cost per Engagement

### Frecuencia Revisión
- **Días 1-3:** Revisión diaria
- **Día 4-7:** Cada 2 días
- **Semana 2+:** 2-3 veces por semana

### Criterios Pausa/Optimización
**Pausar si:**
- CPM > 30 R$ sostenido 3+ días
- CTR < 0,5% sostenido 3+ días
- Frecuencia > 3,5 sostenido 3+ días
- Presupuesto agotado

**Optimizar si:**
- Una variante copy claramente mejor → pausar otras
- Público saturado → ampliar intereses o geografía
