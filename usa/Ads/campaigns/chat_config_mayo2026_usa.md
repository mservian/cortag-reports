## 📝 NOTAS

- El botón **"Find a dealer near me"** y **"Where can I buy Cortag?"** son prioritarios — alineados con CTA de los anuncios
- **Online sí existe** pero varía por estado y no siempre interesa dirigirles a sitios concretos — la respuesta pide ubicación para que el equipo decida qué recomendar caso a caso
- **No nombrar retailers específicos** (ej. Contractors Direct) en las respuestas automáticas — dejar que el equipo USA lo gestione manualmente según el lead
- El botón **"I want to know a price"** captura producto + estado → pasar a Juan Luis / comercial USA para seguimiento — no gestionar desde Marketing
- El seguimiento no compromete tiempo exacto de respuesta (depende equipo USA / Juan Luis)
- Revisar con Juan Luis si el equipo USA puede gestionar el volumen de chats entrantes
